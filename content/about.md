## クオ

プログラマーじゃない。仕事はまったく別のことをしている。コーディングは完全に趣味だ。

AIは数年前から使っていたが、いわゆるチャットで「コード書いて」と頼むだけだった。出てきたコードをコピペして動かして、壊れて、また聞いて、また壊れて。その繰り返し。

2026年2月、VS CodeでAIコーディングツール（GitHub Copilot）が使えることに気づいた。これがターニングポイントだった。毎日何時間もコーディングするほどハマって、そこからCopilot → Cursor → Claude Codeと渡り歩き、今は**VS Code + Claude Code + MAXプラン**に落ち着いている。

開発スタイルはアーキテクト型。設計と方針決めは自分でやって、コードの実装はAIに任せてレビューする。コードを直接手で書くタイプではない。

### 作ったもの

- **OLTranslator** — 画面上の外国語テキストをOCRで読み取り、日本語訳をオーバーレイ表示するWindowsアプリ
- **LiveTR** — PCで再生中の動画の英語音声をリアルタイム認識し、日本語字幕表示と読み上げを行うアプリ
