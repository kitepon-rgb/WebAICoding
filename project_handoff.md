## プロジェクト概要

「Claude Code 始めました」は、Claude MAX ユーザーが実際の開発で使いながら学んだことを記録していく技術ブログ。

### ホスティング

- **GitHub Pages** + **Hugo** で構築
- 記事はMarkdownで書き、`git push` で公開
- 集客はX（Twitter）での発信 + Zennへの転載で補う

### サイト名

**Claude Code 始めました**

---

## 著者のプロフィール・開発スタイル

- アーキテクト型：設計と方針決めを自分でやり、コード実装はAIに任せてレビューする
- 画像共有が多い：UIスクリーンショットやエラー画面をAIに見せて質問するスタイル
- VS Code + Claude Code拡張 + MAXプラン（5x）が現在のメイン環境
- コードを直接手で書くタイプではない
- Pythonでアプリケーション開発をしている

---

## ブログの方針

- **実体験ベース**で書く。使ってみて良かったこと、ダメだったことをそのまま書く
- **SEO目的のスカスカ記事は書かない**。画像ばかりでかいサイトとは一線を画す
- 公式ドキュメントの翻訳ではなく、「使ってるけど実は知らなかった」層に向ける
- 著者の口調はカジュアル。「俺」「〜じゃないすかね」くらいの温度感
- 日本語向け
